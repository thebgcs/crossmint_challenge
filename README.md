# Crossmint Challenge

# Installation
### Setup:

$ gh repo clone thebgcs/crossmint_challenge
$ cd crossmint_challenge
$ npm i

### Be sure you have installed:
- [NodeJs](https://nodejs.org/en/)
- [Npm](https://docs.npmjs.com/)
- [Git](https://git-scm.com/)
- [AXIOUS](https://axios-http.com/docs/intro)

# Execution

**$ npm run start**
